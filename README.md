Tetris Game — Project Documentation

📖 What Is This Project?

This project is a Tetris game — the classic falling-blocks puzzle game — built completely from scratch using Python and a graphics library called Pygame.

🕹️ Controls (How to Play)

.Key                                                                                       What It Does
Left Arrow                                                            Move the falling piece left
Right arrow                                                          Move the falling piece left
Down arrow                                                         Speed up the fall (soft drop)
Up the arrow                                                       Rotate the piece
Spacebar                                                             Instantly drop the piece to the bottom
P                                                                             Pause or unpause the game
R                                                                            Restart the game (only after the game)






⭐ Scoring Guide
Achievement                                        Points  
Clear 1 line                                            100*your current level
Clear 2 lines together                       300*your current level
Clear 3 lines together                       500 × your current level
Clear 4 line together                         800 × your current level

🧩 Features Included

 ✅ All 7 classic Tetris shapes (I, O, T, S, Z, J, L) 
 ✅ Fair random piece generator (no annoying repeats or unlucky streaks) 
 ✅ Ghost piece — shows exactly where your piece will land   ✅ "Next piece" preview panel 
 ✅ Score, level, and lines-cleared tracker 
 ✅ Increasing difficulty as you level up 
 ✅ Pause and restart support 
 ✅ Game Over screen with restart option
